
* _Debes clonarlo en tu HOME de usuario normal para funcionar_

```sh
  git clone https://github.com/tony23x/Predator-Theme.git
```

* _Entra a la herramienta:_
```sh
  cd Predator-Theme
```
* _Ejecuta el instalador:_
```sh

```
* _Ejecuta el script:_

```sh
    bash theme.sh
```
